EJECUTABLE OCTAVE
1- Recomendado abrir el Octave en modo administrador.
2- El programa utiliza la librería Symbolic (https://octave.sourceforge.io/symbolic/) ya incluído en la misma carpeta que el ejecutable
3- El programa tiene la capacidad de ingresar una función en un dominio de x para el cálculo de la serie de Fourier.
4- El gráfico puede mostrar sucesivamente las sumas parciales o el gráfico final.
5- Se puede optar por solo calcular cosenos, senos o la totalidad de la serie.
6- Se puede ingresar los parámetros directamente de la serie de Fourier para graficarla.
